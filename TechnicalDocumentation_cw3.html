Introduction

Python is a popular high-level programming language known for its readability and versatility.

It supports multiple programming paradigms and has a large standard library.

print("Hello, World!")
  • Easy to learn
  • Extensive libraries
Syntax

Python uses indentation to define code blocks instead of braces.

Its syntax is clean and straightforward, making it beginner-friendly.

if x > 0:\n print("Positive number")
  • No semicolons
  • Colon after conditionals
Variables

Variables in Python are created when first assigned a value.

They do not require an explicit declaration to reserve memory space.

x = 5
  • Dynamically typed
  • Case-sensitive
Functions

Functions are defined using the def keyword in Python.

They help organize code into reusable blocks.

def greet(name):\n return "Hello " + name
  • Can have default arguments
  • Support for variable-length arguments
Modules

Modules in Python are files containing Python code.

They can define functions, classes, and variables that you can reuse in other files.

import math\nprint(math.sqrt(16))
  • Standard library modules
  • Third-party modules (like NumPy)
